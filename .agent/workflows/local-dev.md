---
description: Cómo ejecutar el servidor de desarrollo local para pruebas
---

Para ejecutar el sistema en tu entorno local y probar los cambios en tiempo real, sigue estos pasos:

1. Asegúrate de estar en la raíz del proyecto.
// turbo
2. Ejecuta el servidor de desarrollo local:
```powershell
npx live-server
```

Una vez ejecutado, el navegador se abrirá automáticamente. Cualquier cambio que realices en el código se verá reflejado instantáneamente sin necesidad de recargar manualmente.
